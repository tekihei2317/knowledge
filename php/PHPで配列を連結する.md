# PHPで配列を連結する

## array_merge

キーに数字と文字列が混ざっているとややこしい...

同じキーがある場合は後者で上書きされるっぽい。

## スプレッド演算子

```php
$a = [3, 1, 4];
$b = [1, 5, 9, 2];

$c = [...$a, ...$b];
// [3, 1, 4, 1, 5, 0, 2]

// 配列じゃなくても結合出来る
$d = [3, 1, 4, ...$b];
// [3, 1, 4, 1, 5, 0, 2]
```
