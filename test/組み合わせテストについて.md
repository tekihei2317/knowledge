# 組み合わせテストについて

入力値の色々な組み合わせを試すテスト。テストケースを作成する方法には、直行表やペアワイズ法（オールペア法）などがある。

## ペアワイズ法について

任意の2つの入力項目の組み合わせが全て含まれるように、テストケースを作成するアルゴリズム。例えばラーメンの構成要素として「麺」、「スープ」、「具」があった場合、「麺」×「スープ」、「麺」×「具」、「スープ」×「具」の全ての組み合わせが含まれるようなテストケースを作成する。

入力項目と候補

| 項目   | 候補                          |
| ------ | ----------------------------- |
| 麺     | 極細麺,細麺,中太麺,太麺       |
| スープ | 醤油,味噌,塩,トンコツ         |
| 具     | チャーシュー,ねぎ,海苔,メンマ |

ペアワイズ法で作成したテストケースの例

|     | 麺     | スープ   | 具           |
| --- | ------ | -------- | ------------ |
| 1   | 極細麺 | 醤油     | チャーシュー |
| 2   | 極細麺 | 味噌     | ねぎ         |
| 3   | 極細麺 | 塩       | 海苔         |
| 4   | 極細麺 | トンコツ | メンマ       |
| 5   | 細麺   | 醤油     | ねぎ         |
| 6   | 細麺   | 味噌     | チャーシュー |
| 7   | 細麺   | 塩       | メンマ       |
| 8   | 細麺   | トンコツ | 海苔         |
| 9   | 中太麺 | 醤油     | 海苔         |
| 10  | 中太麺 | 味噌     | メンマ       |
| 11  | 中太麺 | 塩       | チャーシュー |
| 12  | 中太麺 | トンコツ | ねぎ         |
| 13  | 太麺   | 醤油     | メンマ       |
| 14  | 太麺   | 味噌     | 海苔         |
| 15  | 太麺   | 塩       | ねぎ         |
| 16  | 太麺   | トンコツ | チャーシュー |

全ての候補を試す場合は`4 * 4 * 4 = 64通り`になるが、ペアワイズ法を使うと`16通り`まで減らすことが出来た。
